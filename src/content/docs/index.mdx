Welcome to **AstroDocs**, a modern documentation template built with Astro, Tailwind CSS v4, and MDX.

## Quick Start

Get up and running in minutes:

```bash
# Clone the template
bun create astro@latest my-docs -- --template minimal

# Install dependencies
bun install

# Start development server
bun run dev
```

## Features

- **Lightning-fast** static site generation with Astro
- **MDX support** with custom components (callouts, tabs, steps, and more)
- **Dark/light mode** with system preference detection
- **Full-text search** powered by Pagefind
- **Responsive** three-column docs layout
- **Syntax highlighting** with Expressive Code
