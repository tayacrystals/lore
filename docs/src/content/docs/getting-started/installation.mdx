## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** version 18 or higher
- **npm**, **pnpm**, or **bun** as your package manager

## Setup

<Steps>
  <Step title="Create a new Astro project">
    Start by creating a new Astro project:

    <CodeTabs id="create-project">
      <Tab label="npm">
        ```bash
        npm create astro@latest my-docs
        ```
      </Tab>
      <Tab label="pnpm">
        ```bash
        pnpm create astro@latest my-docs
        ```
      </Tab>
      <Tab label="bun">
        ```bash
        bun create astro@latest my-docs
        ```
      </Tab>
    </CodeTabs>
  </Step>

  <Step title="Install Lore">
    Install the Lore integration:

    <CodeTabs id="install">
      <Tab label="npm">
        ```bash
        npm install @tayacrystals/lore
        ```
      </Tab>
      <Tab label="pnpm">
        ```bash
        pnpm add @tayacrystals/lore
        ```
      </Tab>
      <Tab label="bun">
        ```bash
        bun add @tayacrystals/lore
        ```
      </Tab>
    </CodeTabs>
  </Step>

  <Step title="Configure Astro">
    Update your `astro.config.mjs` to use the integration:

    ```js
    // astro.config.mjs
    import { defineConfig } from "astro/config";
    import lore from "@tayacrystals/lore";

    export default defineConfig({
      integrations: [
        lore({
          title: "My Docs",
          sidebar: {
            guides: { label: "Guides", order: 1 },
          },
          social: {
            github: "https://github.com/your-org/your-repo",
          },
        }),
      ],
    });
    ```
  </Step>

  <Step title="Set up content collection">
    Create `src/content.config.ts`:

    ```ts
    import { defineCollection } from "astro:content";
    import { docsLoader } from "@tayacrystals/lore/loaders";
    import { docsSchema } from "@tayacrystals/lore/schema";

    const docs = defineCollection({
      loader: docsLoader(),
      schema: docsSchema(),
    });

    export const collections = { docs };
    ```
  </Step>

  <Step title="Create your first page">
    Create `src/content/docs/index.mdx`:

    ```mdx
    ---
    title: Welcome
    description: My documentation site
    order: 0
    ---

    Hello, world!
    ```
  </Step>

  <Step title="Start the dev server">
    Run the development server:

    ```bash
    npm run dev
    ```

    Open [http://localhost:4321/docs](http://localhost:4321/docs) in your browser.
  </Step>
</Steps>

<Callout type="note">
  Lore automatically sets up Expressive Code, MDX, Sitemap, Pagefind, and Tailwind CSS for you. No need to install or configure them separately.
</Callout>

## Project Structure

After setup, your project should look like this:

<FileTree>
- src/
  - content/
    - docs/ Your documentation pages
  - **content.config.ts** Content collection config
  - pages/ Additional pages (optional)
- public/ Static assets
- **astro.config.mjs** Astro configuration
- package.json
</FileTree>
