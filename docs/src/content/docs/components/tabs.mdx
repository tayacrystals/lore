import { Tabs, Tab, Callout } from "lore/components";

Organize alternative content into switchable panels:

<Tabs>
  <Tab label="React">
    React uses a virtual DOM and JSX syntax. Components are the building blocks of any React application.
  </Tab>
  <Tab label="Vue">
    Vue uses a template-based syntax with a reactivity system. Single-file components keep template, script, and style together.
  </Tab>
  <Tab label="Svelte">
    Svelte compiles components at build time, resulting in minimal runtime overhead and smaller bundle sizes.
  </Tab>
</Tabs>

```mdx
<Tabs>
  <Tab label="Option 1">
    Content for option 1...
  </Tab>
  <Tab label="Option 2">
    Content for option 2...
  </Tab>
</Tabs>
```

<Callout type="note">
  Tab selections are persisted in localStorage across page navigations.
</Callout>

<Callout type="tip">
  For code blocks, use [`CodeTabs`](/docs/components/code-tabs) instead â€” it integrates tabs directly into the code block header for a cleaner look.
</Callout>

## Props

### Tabs

| Prop | Type     | Default | Description                          |
| ---- | -------- | ------- | ------------------------------------ |
| `id` | `string` | random  | Key for localStorage tab persistence |

### Tab

| Prop    | Type     | Default | Description    |
| ------- | -------- | ------- | -------------- |
| `label` | `string` | ---     | Tab button text |
