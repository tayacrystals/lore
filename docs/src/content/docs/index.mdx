## Quick Start

Get up and running in minutes:

```bash
# Create a new Astro project
npm create astro@latest my-docs

# Install Lore
npm install lore

# Start development server
npm run dev
```

## Features

- **Zero-config setup** with a single Astro integration
- **MDX support** with custom components (callouts, tabs, steps, and more)
- **Dark/light mode** with system preference detection
- **Full-text search** powered by Pagefind
- **Responsive** three-column docs layout
- **Syntax highlighting** with Expressive Code
