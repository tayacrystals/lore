# Bienvenue dans Lore v1

Ceci est la documentation v1 en français.

## Pour Commencer

Bienvenue dans l'exemple complexe démontrant à la fois le versionnage et l'internationalisation.

## Fonctionnalités

- **Versionnage**: Changez entre v1 et v2
- **Internationalisation**: Changez entre anglais et français
- **Combiné**: Les deux fonctionnalités fonctionnent ensemble

## Navigation

Utilisez les sélecteurs de version et de langue dans l'en-tête pour explorer différentes versions et langues.
