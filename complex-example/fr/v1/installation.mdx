# Guide d'Installation

## Prérequis

Avant d'installer Lore, assurez-vous d'avoir :

- Node.js 18+
- Bun (optionnel mais recommandé)

## Installation

### Utiliser Bun (Recommandé)

```bash
bun install -g lore
```

### Utiliser npm

```bash
npm install -g lore
```

### Utiliser pnpm

```bash
pnpm add -g lore
```

## Démarrage Rapide

1. Créez un nouveau répertoire de documentation :

```bash
mkdir my-docs
cd my-docs
```

2. Initialisez votre documentation :

```bash
lore init
```

3. Démarrez le serveur de développement :

```bash
lore dev
```
