# Installation Guide

## Prerequisites

Before installing Lore, make sure you have:

- Node.js 18+
- Bun (optional but recommended)

## Installation

### Using Bun (Recommended)

```bash
bun install -g lore
```

### Using npm

```bash
npm install -g lore
```

### Using pnpm

```bash
pnpm add -g lore
```

## Quick Start

1. Create a new docs directory:

```bash
mkdir my-docs
cd my-docs
```

2. Initialize your docs:

```bash
lore init
```

3. Start the dev server:

```bash
lore dev
```
