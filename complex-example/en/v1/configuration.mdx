# Configuration

## Basic Configuration

Create a `lore.yml` file in your docs directory:

```yaml
title: My Documentation
description: A description of your docs
color: blue
```

## Versioning

Enable versioning in your config:

```yaml
versioning: true
defaultVersion: v2
```

Then create version directories:

```
docs/
  v1/
    index.mdx
  v2/
    index.mdx
```

## Internationalization

Enable internationalization:

```yaml
internationalization: true
defaultLocale: en
```

Create locale directories:

```
docs/
  en/
    index.mdx
  fr/
    index.mdx
```

## Translation Links

Link translated pages using `translation-of`:

```yaml
---
translation-of: installation
---

# Guide d'Installation
```

## Combined Features

When using both features together:

```yaml
versioning: true
internationalization: true
defaultVersion: v2
defaultLocale: en
```

Structure your docs as:

```
docs/
  en/
    v1/
    v2/
  fr/
    v1/
    v2/
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `title` | string | "Docs" | Site title |
| `description` | string | - | Meta description |
| `color` | string | "#1d4ed8" | Accent color |
| `links` | array | - | Header links |
| `plugins` | array | - | Enabled plugins |
| `versioning` | boolean | false | Enable versioning |
| `defaultVersion` | string | "latest" | Default version |
| `internationalization` | boolean | false | Enable i18n |
| `defaultLocale` | string | "en" | Default locale |
